# 💰 Projeto 01 - Controle de Gastos

## Objetivo

Criar um programa capaz de controlar gastos mensais a partir do salário informado pelo usuário.

---

## Funcionalidades

- Informar salário
- Cadastrar despesas
- Calcular total de gastos
- Calcular saldo restante
- Exibir um resumo organizado

---

## Conceitos praticados

- Variáveis
- Listas
- Listas compostas
- Laço `for`
- `append()`
- Formatação com `:.2f`
- f-strings

---

## Aprendizados

Durante este projeto pratiquei pela primeira vez a manipulação de listas compostas, aprendendo a armazenar e acessar múltiplos dados em uma mesma estrutura.

Também aprendi a realizar cálculos acumulativos utilizando laços de repetição.

---

## Melhorias futuras

- Classificar gastos por categoria
- Excluir despesas
- Editar despesas
