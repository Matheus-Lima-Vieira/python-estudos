# 📋 Projeto 04 - Lista de Tarefas

## 📖 Sobre o projeto

Este projeto foi desenvolvido durante minha jornada de aprendizado em Python.

O objetivo foi criar um sistema de gerenciamento de tarefas executado pelo terminal, permitindo adicionar, listar e remover tarefas por meio de um menu interativo.

Além da implementação das funcionalidades, o foco deste projeto foi praticar estruturas de repetição, listas, validações e melhorar a organização da lógica do programa.

---

## 🚀 Funcionalidades

- ➕ Adicionar uma ou mais tarefas
- 📋 Listar todas as tarefas cadastradas
- 🗑️ Remover tarefas pelo número da lista
- ✅ Validação para impedir a remoção de tarefas inexistentes
- 🚪 Encerrar o programa

---

## 💻 Tecnologias utilizadas

- Python 3

---

## 📚 Conceitos praticados

- Listas
- `append()`
- `pop()`
- `len()`
- `enumerate()`
- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`while` e `for`)
- Manipulação de strings (`strip()` e `lower()`)
- Validação de entrada do usuário
- Organização de menus interativos

---

## 📷 Exemplo de utilização

```text
==== LISTA DE TAREFAS ====

1 - Adicionar Tarefa
2 - Listar Tarefas
3 - Remover Tarefa
4 - Sair
```

Exemplo da listagem:

```text
==== LISTA DE TAREFAS ====

1 - Estudar Python
2 - Subir projetos no GitHub
3 - SQL
```

Ao remover uma tarefa:

```text
Qual tarefa deseja apagar?
2

Tarefa "Subir projetos no GitHub" removida com sucesso!
```
