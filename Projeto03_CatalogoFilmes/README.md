# 🎬 Projeto 03 - Catálogo de Filmes

## Objetivo

Criar um sistema de catálogo de filmes utilizando Python no terminal.

---

## Funcionalidades

- Adicionar filmes
- Listar filmes cadastrados
- Menu interativo
- Verificar catálogo vazio
- Encerrar aplicação

---

## Conceitos praticados

- Loops `while`
- Estruturas condicionais
- Listas compostas
- Entrada de dados
- Organização de menus
- Estruturação de programas

---

## Aprendizados

Durante este projeto pratiquei bastante a construção de menus interativos e a utilização de listas compostas para armazenar informações relacionadas entre si.

Também foi um excelente exercício para melhorar minha lógica na organização do fluxo do programa.
